# 📚 数学试卷汇编 (Mathematics Exam Compilation)

本项目是一个 LaTeX 编译项目，用于汇编和排版历年数学优化、线性代数、数值计算等相关的考试试卷和习题。

项目结构旨在清晰地分离内容和样式，方便内容的增补和维护。

---

## 📂 项目结构

本项目采用模块化结构，将 LaTeX 样式定义和内容主体分离。

| 文件名 | 类型 | 描述 |
| :--- | :--- | :--- |
| `main.tex` | 主文件 | LaTeX 文档的入口文件，定义了文档结构、宏包、样式和编译配置，并**引用**所有年份的试卷文件。 |
| `20xx.tex` | 内容文件 | **（例如：`2008.tex`, `2010.tex`）** 包含某一特定年份或时间段的所有试题内容。 |
| `README.md` | 文档 | 本项目说明文档。 |

---

## 🛠️ 编译环境与要求

### 1. 编译程序

本项目必须使用 **XeLaTeX** 进行编译，以确保中文字体（如宋体）和数学符号的正确显示。

### 2. 依赖宏包

在 `main.tex` 中引用了以下主要宏包：
* **`ctexrep`**: 支持中文排版的基础文档类。
* **`geometry`**: 页面布局和边距设置。
* **`amsmath`, `amssymb`**: 数学公式排版。
* **`fancyhdr`**: 页眉页脚定制。
* **`fontspec`**: XeLaTeX 字体设置（用于设置宋体和 Times New Roman）。

---

## 🚀 如何使用和编译

### 步骤 1: 准备环境

确保你的系统已安装 **TeX Live** 或 **MiKTeX** 等 LaTeX 发行版，并包含 XeLaTeX 编译器。

### 步骤 2: 编译文档

在终端或你的 LaTeX 编辑器（如 TeXstudio, VS Code + LaTeX Workshop 扩展）中，使用 `xelatex` 命令编译 `main.tex`：

```bash
xelatex main.tex
